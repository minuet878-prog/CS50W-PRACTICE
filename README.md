# CS50W Practice

CS50's Web Programming with Python and JavaScript 課程作業與練習。

---

## Problem Set 0 — Search

Google 搜尋頁面的前端刻板，包含一般搜尋、圖片搜尋與進階搜尋功能。

**使用技術：** HTML, CSS

**功能：**
- Google Search 首頁
- Google Image Search
- Google Advanced Search
- 「好手氣」按鈕

---

## Problem Set 1 — Wiki

簡易的 Wikipedia 風格百科全書網站，支援 Markdown 內容編輯與搜尋。

**使用技術：** Python, Django, HTML, CSS

**功能：**
- 瀏覽與檢視 wiki 條目
- 建立新條目與編輯既有條目
- 關鍵字搜尋（完全比對與部分比對）
- 隨機頁面
- 自製 Markdown → HTML 轉換器（不依賴第三方套件）

## Problem Set 2 — Commerce

eBay 風格的線上拍賣網站，支援商品上架、出價、留言與關注清單。

**使用技術：** Python, Django, HTML, CSS, SQLite

**功能：**
- 商品上架與分類瀏覽
- 即時出價系統（含價格驗證）
- 留言功能
- Watchlist 關注清單
- 拍賣關閉與得標通知
- 使用者註冊、登入、登出
